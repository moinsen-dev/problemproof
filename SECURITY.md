# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities or for reports
that contain personal data.

Send a private report to [developer@moinsen.dev](mailto:developer@moinsen.dev)
with:

- the affected component or URL;
- a reproducible description;
- the potential impact;
- a safe proof of concept, if available.

Do not access, alter, or retain data that does not belong to you. We will
acknowledge a report as soon as practical and coordinate a responsible fix and
disclosure with the reporter.

## Supported version

ProblemProof is currently pre-1.0. Security fixes target the version deployed
at [problemproof.moinsen.dev](https://problemproof.moinsen.dev) and the latest
commit on the default branch.
